# CRUD de Usuários em C# com padrão MVC

Aplicação de console desenvolvida em C# com o objetivo de praticar a implementação de um CRUD (Create, Read, Update, Delete) utilizando o padrão de arquitetura MVC.

## Visão geral
Este projeto simula um sistema simples de gerenciamento de usuários via terminal.  
A aplicação permite criar, visualizar, atualizar e remover registros, mantendo uma separação clara entre dados, interface e lógica de controle.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Atualização de dados
- Remoção de usuários

## Arquitetura
O projeto foi estruturado seguindo o padrão MVC:

- **Model**: define a estrutura dos dados (usuário)
- **View**: responsável pela interação com o usuário via console
- **Controller**: gerencia o fluxo da aplicação e as regras de negócio

Essa separação facilita a manutenção e organização do código.

## Tecnologias utilizadas
- C#
- .NET
